# To-Do List Application

## Description
A simple Python To-Do List application that allows users to:
- Add tasks
- View tasks
- Exit the program

## Technologies Used
- Python

## How to Run
1. Save the code as `todo_list.py`
2. Open Terminal/Command Prompt
3. Run:

python todo_list.py

## Features
- Stores multiple tasks using a list
- Displays tasks using loops
- User-friendly menu system
